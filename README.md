## Project Structure

- `WebApplication1/` - main project folder
  - `BLL/` - Business Logic Layer
  - `DAL/` - Data Access Layer
  - `BOL/` - Business Object Layer
  - `Controllers/` - MVC controllers handling requests
  - `wwwroot/` - front-end pages and static files
- `Docs/` - project documentation with screenshots (PDF)
- `Scripts/` - SQL scripts for database tables
